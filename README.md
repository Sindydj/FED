    </ul>

     <li> De controls zoals knoppen hebben geen focusstates. </li>
        <li> Er is geen skip link aanwezig. </li>
    </ul>
  </li>

<li> Appearance: 
    <ul> 
    <li> Er is geen dark/light mode. Als ik mijn computer op dark-mode zet, veranderd er niks aan de website. </li>

    </ul>
  </li>

<li> Animation:
